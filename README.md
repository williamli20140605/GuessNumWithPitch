# Voice Guessing Game: Singing Pitch (C4 - C5)

A voice-controlled "Guess the Number" game. The computer picks a target from 0 to 10, and you try to match it by singing the correct pitch.

## How to Play
1. **The Target**: The computer chooses a integer between **0** (Middle C) and **10** (Upper C).
2. **Sing to Guess**: Sing at a steady pitch to "input" a whole number.
3. **Feedback**:
   - **TOO BIG**: Your pitch was too high.
   - **TOO SMALL**: Your pitch was too low.
   - **CONGRATS!**: You hit the target!

## Features
- **Steady Pitch Submission**: Hold a note for ~0.8s to submit your guess.
- **Dual Platform**:
  - **Web App**: Accessible on any browser with microphone access.

### How to Run (Double-Click)
1. **macOS**: Double-click the `index.html` file in the web folder.
2. **Windows**: Double-click the `index.html` file in the web folder.


## Technologies
- **Web**: Web Audio API, JavaScript.
